# 特别说明

后端基础组件框架/平台基于本人2016开始的个人项目, 后续逐步应用到公司项目中, 经过四次大版本升级重构, 其中包含大量基础性代码，其功能可能与其他第三方开源项目工具有很多相似之处，
因大量应用到公司项目中所以才保留，具体定制开发过程中可按需求使用。

# 运行环境

* JDK17+
* Maven3.8+
